KuroganeHammer.com is a website dedicated to cataloguing frame data and hitbox data for Super Smash Brothers for Wii U and Dragon Ball FighterZ made by Kurogane Hammer

This app was made for people to have access to this data from their Android devices without relying on an Internet connection, a problem that might occur on venues or on places without public Wi-Fi

The app allows connections to KH API to obtain new frame data updates without relying on apk releases

App source code is available at https://github.com/rubendal/KHAndroidApp
